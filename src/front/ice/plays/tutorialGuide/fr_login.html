Create the Login Page

You must login to the ICE Server before you can access any pages held under its control. Here is an example of a login page where web users can create for themselves an account by leaving the entry fields blank:

File: play_login.html

This page sets the value of the hidden variable ii_action to "connect", collects the account name and password from the user, and passes them all to the page plays[play_home.html] when the user presses the connect button: 
<FORM ACTION="/ice-bin/oiice.dll/plays[play_home.html]" METHOD="POST">
<INPUT TYPE=hidden NAME="ii_action" value="connect">
...
<INPUT TYPE="IMAGE" BORDER=0 NAME="connect" SRC="/iceimages/logout.gif" ALT="Connect">
If the user has no account, the login will fail and the failure action (specified by ii_error_url) takes them to the page plays[play_autoUser.html]. This page allows the user to create a new account. We will look at how to do that later. First we will take a quick look at the home page for our system.
