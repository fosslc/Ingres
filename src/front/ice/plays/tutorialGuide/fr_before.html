Before You Start

We assume a machine name of globe.

Make sure you have successfully installed the following:

  • Web server
  • Advantage Ingres
  • Advantage Ingres ICE server
  • Advantage Ingres Web Deployment Option tutorial
Finally ensure that you know how to address your ICE server (eg http://globe/ice-bin/oiice.dll)
