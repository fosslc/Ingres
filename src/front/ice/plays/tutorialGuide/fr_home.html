Create the Home Page

You must login to the ICE Server before you can access any pages held under its control.

This page demonstrates the Web Deployment Option INCLUDE feature, which allows the page author to incorporate generic Web Deployment Option code on many pages. An example of its use would be to ensure that each page had a uniform style across the site. In our case, we use the INCLUDE file for another reason, that of including Web Deployment Option code to ensure any open transactions are closed. There is little else that is new other than a list of links to the various pages that demonstrate various HTML or Web Deployment Options features.

The line that includes the extra code is: 

<!-- #ICE REPEAT INCLUDE=`plays[play_TxnCndCmt_h.html]` -->

The syntax for specifing an include file is the same as for when one is linked to. Here the location is plays and the file name is play_TxnCndCmt_h.html. We need to specify the REPEAT keyword because there are Web Deployment Option macro statements to be evaluated in this file.
