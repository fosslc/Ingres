airports(country,region) → Array

Document-method: airports

Returns a list of airports based on a search of a country and region using a row producing procedure. To list all airports whose country code is ‘ES’, specify region as ’%’. The results from the procedure are treated like a SELECT from a table or a view.

connect() → instance_variable

Document-method: connect

Connects to the specified Ingres _database one of 3 ways:

  • If a hostname is defined connect using the form

** @hostname,tcp_ip,listen_address:[user,password]::database

  • If no hostname is available, but there is a username and password, connect to the local database with the supplied credentials
  • Connect to the database as the current user
countries() → Array

Document-method: countries

Provide a list of countries stored in the country table using the SQL query: 

  <tt>SELECT ct_name, ct_code FROM country ORDER BY ct_name</tt>

The countryCode can then be used to get a list of aiports for that country with the regions method.
