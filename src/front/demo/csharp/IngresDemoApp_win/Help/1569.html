How You Load the Airport Code Combo Box Control

When the user selects a different region, the event handler regionComboBox_SelectedIndexChanged is invoked. The selected value from the Region combo box is used to retrieve the list of airports in the region. The results of this query are bound to the Airport combo box, and then displayed.

To load the Airport Code combo box control:

  1. Add an IngresDataAdapter control to the form. See Add an IngresDataAdapter Control.
  2. Construct the query.
  3. Generate a DataSet.
  4. Bind the combo box control to the DataSet.
  5. Populate the DataSet.
