Bind the DataGridView Control to the Route DataSet

Create a new binding source, routeBindingSource, and assign it to the dataGridView1.DataSource property.

Assign the DataSet to the routeBindingSource.DataSource property to bind the results to the dataGridView1 control.

BindingSource routeBindingSource = new BindingSource();

dataGridView1.DataSource = routeBindingSource;

routeBindingSource.DataSource = routeDataSet1;

routeBindingSource.DataMember = "route";
