Create a Profile

The Profile function lets you add personal details that are used in the application.

The Email Address field uniquely identifies an entry.

To add an profile

  1. To clear the fields in the form, click New.
  2. Fill in the fields on the form and click Add.

To modify a profile

  1. Click Modify.
  2. In the Email Address field, select the profile you want to modify.
  3. Fill in the fields on the form and then click Change.

To delete a profile

  1. If the profile you want to delete is not displayed, click Modify.
  2. Select the profile from the Email Address selection list that you want to delete and click Remove.
